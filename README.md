## Вывод
В ходе выполнения лабораторной работы я изучила создание сложных макетов с использованием Flexbox для горизонтальной прокрутки категорий и CSS Grid для построения адаптивной сетки видео, освоила продвинутые техники стилизации: оформление карточек с правильным соотношением сторон, наложение метки длительности на превью, ограничение заголовков несколькими строками и создание плавных hover-эффектов (подъём карточки, масштабирование изображения и тень). Также я приобрела практические навыки адаптивной вёрстки с помощью медиа-запросов под планшеты и мобильные устройства, научилась работать с Git для фиксации этапов разработки и оформлять документацию проекта в README. Этот проект позволил мне понять, как современные веб-интерфейсы строятся из переиспользуемых компонентов и подстраиваются под любые экраны.
